# EphemChat

EphemChat is a simple ephemeral peer-to-peer chat built with WebRTC.

No accounts. No servers storing messages.
Just create a room, share the code, and chat directly.

## Features

* Peer-to-peer messaging using WebRTC
* No signup or registration
* Rooms protected with passwords
* Messages disappear when you close the tab
* Open source

## How it works

EphemChat uses WebRTC for direct browser-to-browser communication and STUN servers for NAT traversal.
